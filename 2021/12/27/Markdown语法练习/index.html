MarkDown 语法练习

前后看了网上很多种版本的 MarkDown 语法教学，总结写来如下，「纯背诵手打」。

标题部分

#+空格 表示一级标题； ##+空格表示二级标题；###+空格表示三级标题…依次类推，最多支持 6 级标题。

例子：

这是一级标题

这是二级标题

这是三级标题

…

单词修饰部分

包括加粗，斜体, 删除线， 粗斜体。
表示斜体， 表示加粗，表示粗斜体，~~表示删除线

例子：
加粗
斜体
粗斜体
中划线

文章修饰符号

我把引用，分割线划分为文章的修饰部分.
‘>’ + 空格 表示引用，’>>’可表示多层引用； ***或—表示分割线

例子：

这是示例引用

这是示例嵌套引用

这是示例分割线

图片和链接

图片与链接也是常用的模块。比如在本文档同级目录下建立一个img的文件夹，利用’/img/图片名称’即可
‘!’ +[图片名称]+(图片链接)：表示图片。
[链接名称]+(url): 表示链接

例子：
图片：
链接：我的博客

列表

列表分为有序列表和无序列表
‘- ‘, ‘+’, 或’*’ 加空格，表示无序列表；
‘1.’加空格，表示有序列表。
前面加两个空格可实现嵌套，也可使用 Tab 键;

例子：

  • 这是一级无序列表
    • 这是二级无序列表
      • 这是三级无序列表
        • 这是四级无序列表
  1. 这是一级有序列表
    1. 这是二级有序列表
      1. 这是三级有序列表

代码区块

只要简单地缩进 4 个空格或是 1 个制表符就可以
使用格式：
```语言类型
代码内容
``` 可以实现代码智能高亮。

例子：
这是普通的文字。

这是代码区块文字。

这是高亮代码块。
//这是java 代码块
public class Demo
{
  public static void main(String[] args)
  {
    System.out.printLn("Hello, I'm Demo!");
  }
}

表格

Markdown 制作表格使用 | (|后有一位空格)来分隔不同的单元格，使用 - 来>分隔表头和其他行
语法格式如下：
‘| 表头 | 表头 |
‘| —- | —- |
‘| 单元格 | 单元格 |
‘| 单元格 | 单元格 |
中间—-表示对齐方式：–：右对齐；:– 左对齐;
:–: 居中对齐;

例子：

左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格

后记

暂时先记这么多，满足日常使用没问题了。
也可参考 菜鸟教程 - MarkDown 使用教程 了解更多技巧。
