Markdown 语法练习

一、标题

markdown 用#号加空格标记，可表示1-6级标题
如：
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二、字体

星号与下划线都可以，单是斜体，双是粗体，三是粗斜体
如：
这是斜体
这是粗体
这是粗斜体

三、换行

  1. 直接在一句话后敲两个空格
  2. 两句话之间加一个空行
  3. 在编辑时想在一行文字显示时在指定位置换行，就在其后加

    如：
    一行文字要在这里
    换行显示

四、引用

  • 引用通过符号 > 来实现, > 符号后空格，可有可无。
  • 引用内可以使用换行符，要结束引用，需要一行空白行。
  • 引用可以嵌套使用。
    如：

    这是一个引用

    这是一个引用的引用

    这是一个引用的引用的引用

五、链接

链接有两种方式

  1. `链接名称
  2. `[链接名称]<链接地址>
    如：
    这是圆括号方式链接地址
    这是尖括号方式链接地址

六、图片

图片的使用方式是
![图片描述](图片地址)
如：

如果要改变图片位置大小，可以用
![图片描述](图片地址#pic_center空格=长×宽)
如：
![本地](D:\Kay\Pictures\头像\SeaThink.jpg#pic_center =30×30)

七、列表

列表分为有序列表和无序列表

  • 有序列表：数字加上 . 号，再加一个空格
  • 无序列表：使用 * 或 + 或 - ， 再加一个空格
    如：
    • 加号无序列表
    • 减号无序列表
    • 星号无序列表
  1. 数字有序列表1
  2. 数字有序列表2

若要控制列表的层级，则要在列表符号前加tab
如：

  • 星号无序列表
    • 星号二级无序列表

八、分割线

使用分割线，可以用三个连续的 - 或 + 号，注意分割线上须空一行！！
如：
下面是一条分割线

九、删除线

在要添加删除线的文字前后添加两个 ~~
如：
这是一段加删除线的文字

十、下划线

与HTML语法类似，在要添加下划线的文字前与后加<u>和</u>
如：
这是一段加下划线的文字

十一、代码块

  • 若要在一行内引用代码，只需要将代码部分用反引号 ` 引起来就可以。
    如：
    这是一段code和文字

  • 若要在一个块内引用代码，则在需要引用的代码块的前一行和后一行使用三个反引号`， 同时可在前一个反引号后写入代码的语言。
    如：

    //这是一块java 代码
    public class Demo {
    	public static void main(String[] args) {
    		System.out.println("Hello world!");
    	}
    }

    支持如下语言：

十二、表格

表格使用|来分割不同的单元格，使用-来分割表头和其他行。

  • :- : 将表头及单元格内容左对齐
  • -: : 将表头及单元格内容右对齐
  • :-: : 将表头及单元格内容居中对齐
    如：
    序号（居中） 姓名(左对齐) 年龄(右对齐)
    1 张三 18
    2 李四 30

十三、脚注

一般在论文或其他文献中常能看到脚注，点击脚注标号可以直接跳转到加注位置。
脚注的表达用[^数字]来表示。
如：
这段文字参考了文献一^1和文献二^2

十四、特殊符号

对于markdown中的语法符号，前面加反斜线 \ 即可显示符号本身。
如：

\\
\*
\-
\+
\.
等等
